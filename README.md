## Why Linux?
Linux is the industry standard for security work because:

* **Transparency:** Inspect every process, file, and system behavior.
* **Automation:** Automate tasks, reconnaissance, and exploitation with Bash scripting.
* **Compatibility:** Most security tools (Nmap, Burp Suite, Metasploit) are native to or optimized for Linux.
